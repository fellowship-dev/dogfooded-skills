# Stage 04: Final PR (subagent)

## Inputs
- `.procedure-output/build-train/00-setup/handoff.md` — repo, default branch, build branch, manifest
- `.procedure-output/build-train/02-build-fanout/handoff.md` — per-issue PR + status
- `.procedure-output/build-train/03-merge-chain/handoff.md` — what actually merged

## Pre-condition
If stage 03 merged zero PRs, write handoff with `final_pr: none (nothing merged)` and stop — the
orchestrator will emit the failure marker.

## Task
Open ONE final PR from the build branch to the default branch. This PR carries **no** `build-train`
label, so it enters the normal review pipeline once — the whole point of the train.

## Steps

1. Build the issue/PR summary table from stages 02 + 03 (only merged issues count as Merged).

2. Create the final PR:
```bash
gh pr create --repo $REPO \
  --base $DEFAULT_BRANCH \
  --head $BUILD_BRANCH \
  --title "build-train: $BUILD_BRANCH (N issues)" \
  --body "## Build Train

N issues implemented and merged into this branch:

| Issue | Title | PR | Status |
|-------|-------|----|--------|
| #10 | Brand assets | #50 | Merged |
| #14 | Blog pages | #51 | Merged |

Individual PRs linked above for per-issue review.

---
Generated by build-train"
```

3. Confirm the final PR has NO `build-train` label (do not add it).

4. Write handoff.

## Output: handoff.md

Path: `.procedure-output/build-train/04-final-pr/handoff.md`

```markdown
# Stage 04: Final PR

## Final PR
number: #{N or none}
base: {DEFAULT_BRANCH}
head: {BUILD_BRANCH}
build_train_label: false   # MUST be false

## Summary
| Issue | Title | PR | Status |
|-------|-------|----|--------|
| #10 | Brand assets | #50 | Merged |
| #14 | Blog pages | #51 | Merged |
| #15 | ... | — | Skipped |
```

## Success criteria
- Exactly one final PR opened, base = default branch, head = build branch
- Final PR has NO build-train label
- Summary table reflects merged vs skipped accurately

## Failure
- Nothing merged (pre-condition) → `final_pr: none`, orchestrator emits failure
- `gh pr create` error → write handoff with the error; orchestrator emits failure
